## Setup & Installation

1. Make sure you have the latest version of Python installed.

2. Clone the repository:
   ```bash
   git clone <repo-url>
   ```

# Install the required dependencies:
```bash
1. pip install -r requirements.txt
```
# Running The App

## Execute the following command to run the application:
```bash
python main.py
```
